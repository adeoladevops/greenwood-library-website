Creating a User on CentOS 7

This guide provides the steps to create a user on CentOS 7, including commands for adding a home directory after the initial creation.

Step 1: Create the User without a Home Directory

Use the useradd command to create the user without a home directory:

sudo useradd -M huaweitag

Step 2: Set the User's Password

Use the passwd command to set the password manually:

sudo passwd huaweitag

You will be prompted to enter and confirm the new password.

Step 3: Set the Account Expiration Date

Use the chage command to set the account expiration date to one week from today:

sudo chage -E $(date -d "+7 days" +"%Y-%m-%d") huaweitag

Step 4: Create the Home Directory (if needed)

If you initially created the user without a home directory and now want to add one, follow these steps:

Create the Home Directory

sudo mkdir /home/huaweitag

Set Ownership and Permissions

sudo chown huaweitag:huaweitag /home/huaweitag
            sudo chmod 755 /home/huaweitag

Update the User's Home Directory

Update the user's home directory in the /etc/passwd file:

sudo usermod -d /home/huaweitag huaweitag

Alternative: Create User with Home Directory Initially

To create a user with a home directory from the start, use the -m option with useradd:

sudo useradd -m huaweitag

This command automatically creates the home directory and sets the correct permissions.

Summary

To create a home directory for an existing user who was created without one, manually create the directory, set the ownership and permissions, and update the user's home directory path. To create a user with a home directory from the start, use the -m option with useradd.
